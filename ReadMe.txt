Library Management System
The source code of the app is written in java using the 
basic concepts of Object oriented Programming.It is a 
Console based app that works to perform various
 functions of library.It stores data in files by the
 use of file handling.
The Main menu for library management system is listed below:

1. Add Book
2. Add User
3. Display Books
4. Display Borrowed Books
5. Search Books by User ID
6. Borrow Book
7. Return Book
8. Exit

User has to give the input and enter required data in order
to perform that specific function.
